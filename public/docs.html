Toys Store API

/toys

# Method Url Request Body Request Query info
1 GET /toys - skip?, s? Give you a list of toys from your collection
2 GET /toys/category/:catname - skip? Give you a list of toys from a specific category
3 POST /toys { "name": "String", "info": "String", "category": "String", "img_url": "String", "price": "Number" } - Create a new toy
4 PUT /toys/:id { "name": "String", "info": "String", "category": "String", "img_url": "String", "price": "Number" } - Update an existing toy by ID
5 DELETE /toys/:id - - Delete a toy by ID
6 GET /toys/prices - skip?, min, max Give you a list of toys within a price range
7 GET /toys/single/:id - - Get a single toy by ID
8 GET /toys/count - - Get the count of total toys
